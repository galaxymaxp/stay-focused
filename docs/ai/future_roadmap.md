3. **Calendar sync feeding the schedule**
   - Strengthen ingestion of deadlines/events into scheduler logic.
   - Keep calendar accurate while preserving schedule-first UX.

4. **A4 printable/PDF reviewer maker from uploaded academic templates**
   - Generate reviewer outputs in school-ready printable formats.
   - Align visual output with familiar study handout structures.

5. **Better Do draft handling for code-related tasks (HTML/CSS/JS sections)**
   - Improve structure and flow for programming/class coding work.
   - Keep draft composition compatible with scheduled execution blocks.

6. **Reviewer UI polish to feel like an auto-generated study handout**
   - Increase readability and academic utility.
   - Reduce cognitive overhead while reviewing.

7. **Professional PDF/DOCX export from Do drafts**
   - Provide polished export quality for submissions and sharing.
   - Ensure formatting consistency across content types.

8. **Claude UI inspection pass for information overload**
   - Run focused heuristics for clutter reduction and hierarchy clarity.
   - Identify high-friction screens where action intent is buried.

9. **Canvas token guide popup/video/animation**

10. **Keep `/docs/ai/handoff.md` updated after each AI coding session**
